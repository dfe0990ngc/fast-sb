## 📋 Overview

FAST-SB is a digital system developed to **streamline the processing, monitoring, and management** of franchise applications and related records within the Sangguniang Bayan of Santa Cruz, Davao del Sur.

By digitizing and centralizing the franchise application workflow, FAST-SB reduces manual paperwork, accelerates processing times, and provides applicants and legislators with real-time visibility into application status — making local governance faster and more efficient.

---

## ✨ Features

- 📡 **Real-Time Status Tracking** — Monitor application progress at every stage of the workflow
- 🗂️ **Records Management** — Centralized storage and retrieval of all franchise-related documents
- 👤 **Role-Based Access** — Separate views and permissions for applicants, staff, and administrators
- 📊 **Reports & Analytics** — Generate summaries and reports on franchise application data
- 📱 **Responsive Design** — Accessible on desktop, tablet, and mobile devices
- 🔒 **Secure Data Handling** — Ensures integrity and confidentiality of application records
